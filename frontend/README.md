# Flowvisior Frontend

React + TypeScript + Vite frontend для Flowvisior.

## Запуск

```bash
cd frontend
npm install --no-audit --no-fund
npm run dev
```

Frontend работает на `http://localhost:5173` и проксирует `/api` на Spring Boot backend `http://localhost:8080`.

## Что изменено в этой версии

- весь интерфейс переведён на русский;
- главный экран теперь `Today`: задачи, привычки, квесты, дедлайны и персонаж в одном месте;
- задачи создаются на главном экране, отдельный пункт меню задач скрыт и редиректит в Today;
- аватар показывается на Today и в профиле, аккуратно вместе с характеристиками;
- подготовлены 3 персонажа, у каждого есть 5 HP-состояний для смены аватара;
- верхнее меню: Today, Привычки, Квесты, Календарь, Профиль;
- Статистика, История и Админка спрятаны в меню «Ещё»;
- действия в Today получают оптимистичное обновление: пункт сразу зеленеет/зачёркивается или помечается проваленным;
- календарь стал компактнее: сегодняшний день ведёт на Today, прошлые и будущие даты открываются как просмотр.
