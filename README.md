# Flowvisior

Проект состоит из Spring Boot backend и React/Vite frontend.

- `backend/` — API, сущности, сервисы и миграции Flyway.
- `frontend/` — интерфейс приложения.

## Быстрый запуск

```bash
cd backend
./mvnw spring-boot:run
```

```bash
cd frontend
npm install --no-audit --no-fund
npm run dev
```
